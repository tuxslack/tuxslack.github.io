* DLC Boot

O DLC Boot, essa super Live CD é completa para fazer manutenção do Windows e de seu PC,
você pode rodar ela direto do DVD ou usar pelo Boot do pen drive. Abaixo você pode 
fazer o download dessa ferramenta. Com ele é possível você repara seu PC, fazer backup 
completo de HDs, clonar sistemas ou discos. 

Reparar erros no HD, dividir, excluir e criar partições em discos rígidos. 
Remover senhas do Windows e muito mais. Ele dá suporte a EFI (UEFI) E BIOS (LEGACY). 
Ou seja, compatível com os novos SETUP de placa mães tanto de notebook quanto PCs desktop.

Você pode quebrar senhas do Windows, clonar sistema, particionar e recuperar HDs, 
testar seu PC, remover vírus, limpar o sistema, testar memórias, recupera Windows e 
muitas outras ferramentas que você só vai descobrir fazendo teste.Funciona em todos 
os PCs com base em processadores x64 bits, são muitos programas que você terá, a 
interface do programa é baseada no Windows 10, sendo otimizado para funcionar com 
muita fluência a fazer operações nesse ambiente.

Antes de você entrar na interface baseada no Windows, você terá um menu que fica a 
disposição para recuperar ou verificar hds com HDD Regererator, Memtest86 6.3 para 
teste de memória, Ghost 11.5 para clonar hd, e outros. Cada um com suas ferramentas 
diferente.

Logo na tela inicial o usuário irá contar com opções avançadas, são apresentadas 
categorias, e em cada categoria são listados programas que vão ajudar nas funções 
especificas que eles podem ajudar. Além dessas opções podemos escolher entrar na 
LIVE CD com interface do Windows Xp, 10 x86/64 bits, nelas todos os programas 
necessários serão listados.

* Clonezilla

O Clonezilla é um programa de partição/clonagem de imagem/clonagem semelhante 
ao Acronis True Image ou ao Norton Ghost., com funcionalidades avançadas e 
bem divulgada. Ele ajuda você a fazer a implantação do sistema, backup 
e recuperação bare-metal. Dois tipos de Clonezilla estão disponíveis, 
Clonezilla live e Clonezilla SE (edição do servidor). Clonezilla live 
é adequado para backup e restauração de uma única máquina. Enquanto o 
Clonezilla SE é para implantação em massa, ele pode clonar muitos 
computadores (mais de 40!) Simultaneamente Clonezilla salva e 
restaura apenas blocos usados ​​no disco rígido. Isso aumenta a eficiência 
do clone. Com alguns hardwares de ponta em um cluster de 42 nós, 
foi relatada uma restauração multicast a uma taxa de 8 GB / min.
