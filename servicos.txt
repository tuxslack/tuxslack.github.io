=> Configuração de Redes

Configuração de Wireless (diretivas de segurança, expansão de sinal, otimização do sinal), instalação de modens, roteadores, compartilhamento de arquivos entre computadores, compartilhamento na nuvem, e outros.



Configuração de Wireless

A configuração de redes wireless permite o acesso à internet sem precisar de um cabo conectado ao computador. Quando configurada corretamente, é possível navegar na web a partir de qualquer equipamento que esteja no alcance da rede.
Tipos de serviços realizados

    configuração de modem/roteador
    senha de segurança ao acesso wireless
    configuração de acesso wireless por endereço MAC
    bloqueio ao acesso de sites específicos no roteador
    limitar a velocidade da internet(controle de banda) por equipamento
    controle de pais ao acesso a internet
    bloquear/abrir portas (TCP/UDP) específicas


Problemas comuns

    não consigo acessar o meu roteador
    a velocidade oscila muito durante o uso da internet
    não é possível se conectar na internet por meio de um notebook/celular
    a internet cai constantemente
    não consigo ver o nome da minha rede ao procurar uma rede para se conectar
    não é possível acessar as configurações do meu roteador
    roteador envia sinal mas não consigo acessar uma página da internet




A configuração de uma rede permite que computadores em diferentes pontos se comuniquem entre si, compartilhando arquivos, programas, impressão, internet e assim por diante. Redes Wireless inicialmente mais caras, estão se tornando mais baratas e fáceis de instalar em relação a redes cabeadas, qual configurar vai depender da necessidade e desempenho esperado. Porém é comum mesclar ambos os tipos. Endereço IP, Máscara de sub-rede, Gateway padrão, servidor dhcp entre outros são alguns dos termos mais comuns utilizados na configuração de redes.




Problemas mais comuns

    não consigo visualizar o computador “x” na rede
    envio um arquivo de rede para imprimir mas nada acontece
    não consigo acessar a internet a partir dos computadores da rede
    É exibido um alerta “conflito de ip” e não acesso a rede
    recebo o erro “acesso negado” e não consigo copiar um arquivo de um computador para o outro
    ao tentar acessar um computador na rede recebo a mensagem de que está inacessível
    Não consigo logar em um computador na rede, recebo a mensagem “falha no logon”






=> Servidores

Configuração de servidores, diretivas de segurança, compartilhamento e backups automatizados, scripts de impressão, controle de acesso e conteúdo


Configuração de Servidores

Uma rede pode ser ponto-a-ponto (peer-to-peer), também chamada como grupo de trabalho ou uma rede do tipo cliente/servidor(client-server). A grande vantagem de uma rede cliente/servidor é o fato de possuir um ou mais computador dedicado(servidores) que centraliza a administração da rede. O servidor é o local central onde é permitido aos usuários compartilhar e acessar os recursos da rede. Realizo toda a configuração inicial dos servidores para que seja possível os computadores acessarem os seus recursos.






=> Manutenção de Computadores


Montagem de computadores e upgrade de componentes, manutenção corretiva e preventiva no computador (pc não liga, computador travando, lento, com barulho, teclado ou mouse não funciona), limpeza de componentes.


* Limpeza Interna do Computador

É realizada a limpeza interna do computador. A limpeza interna do computador deve ser feita periodicamente, auxiliando na redução de temperatura, tempo de vida dos componentes e estabilidade do computador. Durante este procedimento, é importante observar o correto manuseio das peças devido a energia estática. É levado os equipamentos necessários para uma segura e completa limpeza do seu computador.


Problemas mais comuns

    computador apresenta ruído elevado durante uso
    computador desliga sozinho
    computador trava/congela tela aleatoriamente durante o uso
    computador apitando várias vezes durante uso
    computador reinicia sozinho após algum tempo
    computador apresenta tela azul





=> Manutenção e Otimização do Computador

Para o correto funcionamento o Windows deve ser limpo e otimizado em determinados períodos. Com o tempo, o Windows começa a acumular arquivos temporários, programas instalados sem uso, atualizações do windows e de programas, dlls de programas desinstalados enchem o registro, drivers desatualizados e vírus podem surgir durante o uso por diversos fatores. Como resultado o Windows se torna mais lento dificultando a realização de tarefas rotineiras. O técnico realiza a limpeza e otimização do sistema operacional restaurando suas funções




Problemas mais comuns

    computador lento ao ligá-lo
    computador demora ao abrir um programa qualquer
    ao acessar um site o computador fica carregando a página mas não abre
    computador reinicia sozinho ao ligá-lo
    computador apresenta uma janela com erro de arquivo do Windows ausente
    ao acessar um site várias propagandas aparecem na tela do computador
    não é possível remover um programa do windows





=> Desenvolvimento de Sites Profissionais

Desenvolvimento Web em WordPress, Layouts, otimização com o google adwords


Layout

Ao desenvolver um site em wordpress, existem alguns aspectos chaves que devem ser estudados para que o seu site tenha uma boa apresentação. E um deles é o layout, nele você deve gastar um bom tempo estudando o formato, escrita, imagens entre outros aspectos. O layout deve primeiramente valorizar o seu conteúdo, de forma que o foco do seu negócio esteja em contraste. Layouts dinâmicos de uma página, apresentações de trabalho em destaque (portfolio), layouts com scrolling parallax são alguns dos trabalhos realizados.


Problemas mais comuns

    layout não tem um foco no conteúdo do site
    o negócio do cliente é posto em segundo plano, destacando muito o seu trabalho realizado
    as fontes escolhidas não se encaixam na composição e temática do projeto, transmitindo um aspecto pouco profissional do site
    site com o uso de cores excessivas nas fontes apresenta o site de forma negativa
    informações em excesso no site depreciando o layout
    textos confusos e mal posicionados dificultando a sua leitura

=> Backups e Recuperação de Arquivos excluídos

O Backup de seus arquivos é um procedimento indispensável e deve ser feito periodicamente. O backup ou cópia de segurança dos seus dados previne que você perca de forma definitiva informações contidas em seu computador devido a um corrompimento do windows, queda de energia, hd queimado entre outros fatores. O backup deve ser realizado corretamente para que você tenha uma cópia segura e de qualidade dos seus dados. Realizo além do backup a recuperação de seus dados excluídos/deletados acidentalmente ou corrompidos devido a queda de energia.


Problemas mais comuns

    Backup faltando arquivos
    Não consigo acessar o Windows depois de uma queda de energia
    Computador travou e não é possível acessar os dados
    Um documento sumiu da área de trabalho e não está na lixeira
    Deletei uma pasta de documentos e não consigo recuperar
    O backup realizado na HD está corrompido
    Ao descompactar uma pasta de documentos é exibido um erro de corrompimento de arquivos
